Suyash Patil Portfolio Website
Description

This is my personal portfolio website designed to showcase my skills, projects, and contact information. The website has a modern 3D interactive design with a galaxy/starfield background and floating programming language icons that move randomly, giving it a unique and dynamic look. It highlights my work as a computer engineering student and developer.

Features

Responsive Design: Works perfectly on desktops, tablets, and mobile devices.

Hero Section: Includes my profile picture, name, and a brief introduction.

Social Links: Quick access to my LinkedIn, GitHub, and email.

Download Resume Button: Lets visitors download my resume directly.

Skills Section: Highlights my technical skills including HTML, CSS, JavaScript, React, C++, and more.

Projects Section: Displays my main projects like Amazon Clone, Investaura (Gamified Trading Platform), 3D Calculator, and AI-powered Editing Platform.

3D Live Background: Interactive starfield and galaxy-inspired background with moving language icons (Java, C, C++, Python, JavaScript, HTML, CSS, React).

Smooth Scrolling Navigation: Click menu items to scroll to different sections seamlessly.

Technologies Used

Frontend: HTML, CSS, JavaScript

Animations & Effects: CSS animations, Canvas API (for starfield), and DOM manipulation for floating icons.

Fonts & Icons: Google Fonts (Poppins), Font Awesome icons

Responsive Design: Fully mobile-friendly with CSS Flexbox and media queries

How to Use

Clone or download this repository.

Open index.html in a web browser to view the portfolio.

Explore sections: Home, About, Skills, Projects, Contact.
